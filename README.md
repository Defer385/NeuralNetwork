По нему можно понять как утроен главный класс NeuralNetwork
```cpp
NeuralNetwork net;

//Главная функция в начале - .add(...) она дает возможность вставлять слои

```

Теперь можете делать что хотите (в рамках разумного конечно) и выбрать что вам нужно:

```cpp
net.forward(...); //Прямой прогон, принимает и отдает Tensor

net.SLtrain(... , ...); //Классическое 'Обучение с учителем', принимает данные (1 аргумент) и что хотим (2 архгумент), в виде Tensor

net.RLtrain(... , ... , ...); //RL обучение, принимает состояние Tensor, совершенное дейсвтие int, награду double
```

> ВАЖНО: net.RLtrain выполняет "vanilla policy gradient update". Взаимодейсвтие с средой и другими вычислениями происходят благодаря классу Agent который описан ниже

# Про класс Tensor

Класс Tensor является ядром всей системы.

Хотя и данные в Tensor храняться линейно, но Tensor поддерживает до 4-х изменений , и имеет соотвественно 4 функции из группы "чтения" и "записи". Инициализируется он следующим образом

```cpp
Tensor tesnor({10,10}); // Таким образом мы создаем матрицу 10 на 10
Tensor anotherTenxor({3,5,7}); // Таким образом мы создаем тензор 3 на 5 на 7
```

Чтоб получить элементы тензора или записать что мы хотим нужно использовать функции .read # # или .write # #

Для лучшей ориентации по функциям их можно делить как read/write + one/two/three/fout + D (от англ. измерение)

Пример:
```cpp
someTensor.readOneD(x); //Функция вернёт значение из тензора который представлен как линейнный массив
anotherTensor.writeTwoD(3.14, x, y); //На позицию X и Y мы записали (в данном случае) в матрицу число 3.14
 ```

Так же добавлены несколько вспомогательных функций и поддержка работа с std::vector

```cpp
tensor.size(); //Вернёт ЛИНЕЙНЫЙ размер
tensor.config(); //Вернёт вектор в котором расписаны измерения тензора
tensor.clear(); //Заполняет все нулями
tensor.fillRandom(min,max); //Заполняет случайными значениям в желаемом промежутке 
tensor.insertVector({..., ... ... , ...}); //Позволяет вставить линейный вектор, важно чтоб совпадал размер исходного и который вставляем
tensor.dat(); //Возращается ссылку на линейный вектор в котором храняться данные Tensor
tensor.reShape({..., ... ... , ...}) //Позволяет полностью изменить тензор в желаемую конфигурацию. Перенос данных не предусмотрен данной функией
```


# Про обучение с подкрепление

Если мы желаете использовать нейронную сеть для среды, есть готовый класс Agent (агент) который позволит легко его внедрить в вашу игру

Агент работает по A2C, и "критика" с "актером" вы должны инициализировать и вставить самостоятельно:

```cpp
NeuralNetwork actor;
actor.add(new Dense(4, 64, ActivationsType::TANH));
actor.add(new Dense(64, 64, ActivationsType::TANH));
actor.add(new Dense(64, 2, ActivationsType::SOFTMAX));
actor.init(1e-5);

NeuralNetwork critic;
critic.add(new Dense(4, 64, ActivationsType::TANH));
critic.add(new Dense(64, 64, ActivationsType::TANH));
critic.add(new Dense(64, 1, ActivationsType::LINEAR));
critic.init(3e-4);

Agent agent(actor, critic, 0.99); //сначала соотвественно актер, потом агент, а третий аргумент это коэф. гамма
```

Для получение дейсвтия от агента вам достаточно вызвать
```cpp
int action = agent.act(state); //состояние в виде std::vector<double>
```
Для обучения
```cpp
int action = agent.train(state, next_state, action, reward, done); //Состояния тоже в виде в виде std::vector<double>
```
