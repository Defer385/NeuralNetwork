# Что за проект
Проект по воссозданию нейронной сети с нуля.
На данный момент представлена модификация №4, в которой есть так и возможность обучения с учителем, так и обучение с подкреплением

# Пример работы
Для начала требуется инициализировать нейронную сеть.
Пример:
```cpp
#include "NeuralNetwork.h"

int main()
{
    std::vector<double> data(784); 
    
    NeuralNetwork nn({ 784, 256,128,64,10 }, 0.0005, LEAKY_RELU, SOFTMAX);

    std::vector<double> result = nn.forward(data);
} 
```
В данном фрагменте кода представленна нейронная сеть с входным слоем в 784 нейрона, 256 128 64 - кол-во нейроннов для скрытых слоев, 10 - кол-во нейронов для выходного слоя.
    
Коэф. обучения равен 0.0005
    
Функция активации для скрытых слоев LEAKY_RELU
Функция активации для выходного слоя SOFTMAX

# Обучение
Если желаете обучать нейронную сеть методом обучения с учителем, используйте:
```cpp
    nn.SLtrain(data, want);
```
Данные подаются в виде `std::vector<double>, std::vector<double>`

Если использовать метод обучения с подкреплением:
```cpp
    nn.RLtrain(state, action, reward);
```
Данные подаются в виде `std::vector<double>, int, double`


# Что представлено в репозитории?
После сборки `Cmake` и запуска, вы сможете наблюдать за обучением нейронной сети на основе базы данных `MNIST`, обучение занимает ~5-10 минут.

После обучения в консоли вы увитите несколько чисел, и предсказание нейронной сети в процентах.
`Важно - после обучения нейронная сеть на вход получает те цифры на которых НЕ обучалась`
