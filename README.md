# Что за проект
Проект по воссозданию нейронной сети с нуля.
На данный момент представлена модификация №5, в которой кроме многослойного перцептрона (классическая нейронная сеть) была добавлена архитектура CNN (свёрточная нейронная сеть), НО в минимальной конфигурации.
На данный момент в CNN слоях используется только ReLU в качестве функции активации, отстутствуют MaxPool слои, и нету удобной функции для создания CNN слоев. 

# Пример работы
Для начала требуется инициализировать нейронную сеть.
Пример:
```cpp
#include "NeuralNetwork.h"
#include <vector>

int main()
{
    std::vector<double> data(784); 
    
    NeuralNetwork nn({ 784, 256,128,64,10 }, 0.0005, LEAKY_RELU, SOFTMAX);

    std::vector<double> result = nn.forward(data);
} 
```
В данном фрагменте кода представленна нейронная сеть с входным слоем в 784 нейрона, 256 128 64 - кол-во нейроннов для скрытых слоев, 10 - кол-во нейронов для выходного слоя.
    
Коэф. обучения равен 0.0005
    
Функция активации для скрытых слоев LEAKY_RELU
Функция активации для выходного слоя SOFTMAX

# Обучение
Если желаете обучать нейронную сеть методом обучения с учителем, используйте:
```cpp
    nn.SLtrain(data, want);
```
Данные подаются в виде `std::vector<double>, std::vector<double>`

Если использовать метод обучения с подкреплением:
```cpp
    nn.RLtrain(state, action, reward);
```
Данные подаются в виде `std::vector<double>, int, double`

# CNN слои
Если вы желаете добавить CNN слой, вам нужно вручную создать умный указатель

```cpp
    nn.CnnLayers.push_back(std::make_unique<ConvLayer>(28, 28, 8, 4, 3, 1));
```
Тут мы создаем CNN слой, где его конфигурация:
- H входного призрака = 28
- W входного призрака = 28
- Кол-во фильтром (ядер) = 8
- Кол-во входных призраков = 4
- Размер фильтра (ядра) = 3
- Padding = 1

**ВАЖНО**

_Если вы используете CNN слои, то сначала вы добавляете слои, а потом вызываете специальную функция `.checkСompatibility()`, которая проверит совместимость слоев и правильно их инициализирует._

_Так же для прямого прохода вы должны использовать `.MultForward(...)`, а для обучения `.CNN_With_FCN_SL_train(... , ...)`_
